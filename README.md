# Onsen - Pemandian Air Panas 

**Tugas Besar Praktikum Grafik Komputer**  
**Kelompok 4**  
Mata Kuliah: Praktikum Grafik Komputer  
Tahun Ajaran: 2025/2026

## Deskripsi Proyek

Proyek ini merupakan aplikasi 3D interaktif yang mensimulasikan suasana **Onsen** (pemandian air panas) ala Jepang menggunakan teknologi grafika komputer.

Fitur visual & interaksi utama:
- Air dengan efek tembus pandang
- Pencahayaan pada Lampu tanpa pencahayaan global
- Particle uap/air
- Background Malam
- Kontrol kamera bebas (first person)
- Kontrol kamera Orbit
- Interaksi sederhana: buka/tutup pintu

## Teknologi yang Digunakan

- **Bahasa**: C++
- **Graphics API**: OpenGL
- **Window & Input**: FreeGlut
